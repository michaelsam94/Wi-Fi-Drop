## App name (30 chars max)
Wi-Fi Drop

## Short description (80 chars max)
Fast local Wi-Fi file sharing. Send to nearby devices or share via browser.

## Full description (4000 chars max)
Wi-Fi Drop moves photos, videos, documents, and entire folders between phones, tablets, and computers on the same Wi-Fi network—without uploading anything to the cloud.

Pick what to share, find nearby receivers, and transfer at full local-network speed. Need to grab files on a laptop? Turn on Web Share and open the link or scan the QR code in any browser on your network.

**Send**
• Select single files, multiple files, or whole folder trees
• See nearby Wi-Fi Drop receivers on your local network
• Tap a device to start a direct peer-to-peer transfer
• Track progress with speed, percentage, and time remaining

**Receive**
• Broadcast a custom display name so senders can find you
• Accept incoming transfers while the Receive screen is open
• Saved files land in your Downloads/Wifi-Drop folder

**Web Share**
• Host selected files on a local server built into the app
• Share a QR code or URL for quick downloads on a PC or Mac browser
• Handy when the other device is not running Wi-Fi Drop

**History**
• Review completed and in-progress transfers
• See direction (sent or received), file counts, sizes, and timestamps

**Built for local networks**
Wi-Fi Drop uses on-device discovery and direct device-to-device transfers on your Wi-Fi or hotspot. Your files stay on your network—no cloud storage, no account required.

**Permissions**
Location and Nearby Devices access are used for Android Wi-Fi peer discovery on local networks. Storage access lets you pick files to send and save files you receive.
